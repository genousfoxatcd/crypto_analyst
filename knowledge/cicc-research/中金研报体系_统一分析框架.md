## 四、中金行业研究方法论速查

### 行业周期定位框架

```
导入期（<5%渗透率）
  → 主题投资，高弹性，严格止损
  → 代表：人形机器人2025-2027年

成长期（5-30%渗透率）
  → 优选龙头，份额集中逻辑
  → 代表：液冷散热2026-2028年

成熟期（>30%，增速放缓）
  → 高股息/防御性，FCF为王
  → 代表：房地产/电力公用

出清期（产能去化）
  → 等待供需拐点，配置剩者
  → 代表：锂电/光伏2026年
```

### 估值方法适用表

```
消费成长  → PEG / PS
周期化工  → PB / EV/EBITDA  
金融银行  → PB / ROE
科技硬件  → PS / EV/S
电力公用  → 股息率 / EV/EBITDA
房地产    → PB / RNAV
```

## 五、分析框架速查手册

### 中金"四问"行业评估法

```
① 需求在哪里？（市场空间 + 增速）
② 供给谁来提供？（竞争格局 + 壁垒）
③ 政策向不向？（政策支持 or 打压）
④ 值不值这个价？（估值分位 + 风险收益比）
```

### 中金政策传导分析

```
政策信号 → 传导路径 → 效果验证时间 → 受益标的
─────────────────────────────────────────────────
消费以旧换新延续 → 家电/汽车销量 → 3-6个月 → 家电龙头
十五五新兴产业 → 订单/招标 → 12-24个月 → 半导体设备
扩内需 → 社零增速 → 3-6个月 → 消费龙头
```
