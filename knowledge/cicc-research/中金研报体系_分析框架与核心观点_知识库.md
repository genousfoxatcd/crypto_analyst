## 二、中金宏观研究核心框架（2026年）

### 2.1 2026年宏观主轴："增减之间"

**中金2026宏观年度报告标题："增减之间——中国宏观2026年展望"**

**核心含义**：财政、货币、结构政策在扩张与收紧之间寻求动态平衡——既要扩内需，又要控债务；既要稳增长，又要调结构。

**关键判断**：

```
【增长】
- 2026年GDP增长目标：~5%左右
- 出口有望实现6%左右增长
- 内需仍然较弱（消费复苏慢，投资偏弱）
- 劳动密度下降+企业加速出海 → 出口对内需带动边际减弱

【政策】
- 扩内需是重点任务（2024年底中央经济工作会议定调）
- 财政：中央加杠杆，地方稳杠杆
- 货币：配合财政，降准/降息仍有空间
- 消费以旧换新：2026年或继续实施，范围/力度可能优化

【风险】
- 中美AI投资"两极"格局 → 科技竞争加剧
- 地缘政治风险持续上升
- 外部环境变化影响加深
```

### 2.2 2026年两会政府工作报告核心要点

**（2026年3月5日发布）**

| 维度 | 要点 |
|------|------|

## 四、中金行业深度框架（重点赛道）

### 4.1 新能源/锂电

**中金跟踪要点**：
- 产能周期：2024-2025年产能出清进行中，底部渐近
- 供需格局：供给侧去产能 vs 内需政策提振
- 估值：经历大幅回调后，龙头公司性价比显现
- 关注：欧洲新能源车渗透率、美国 IRA 补贴退坡节奏

### 4.2 半导体/AI

**中美AI投资"两极"格局**：
```
美国：AI应用层领先（OpenAI/微软/谷歌）
中国：AI基础设施+制造端优势（华为/百度/阿里）
差异：应用创新 vs 制造效率
```

**中金AI投资框架**：
- 光通信：OFC2026超级周期（迈入光互联时代）
- 算力芯片：国产替代加速（华为昇腾、寒武纪）
- AI应用：垂直行业AI（医疗/金融/制造）

**人形机器人**：
- 宇树科技（Unitree）上市获受理 → 机器人产业化提速
- 2025-2030年：降本曲线陡峭 + 政策密集催化

### 4.3 房地产

**中金判断**：
- 二手房销售面积同比降幅收窄（积极信号）
- 京沪二手房景气度回暖
- 但整体去库存压力仍大
- 政策：防风险 + 去库存双线并进
- 投资建议：龙头国央企 + 资产质量改善主线

### 4.4 消费

**消费以旧换新（2026延续）**：
```
覆盖品类：家电/汽车/家装
优化方向：
- 资金支出节奏（更均衡）
- 补贴渠道（线上+线下）
- 覆盖品类（可能扩大）
```

**中金消费配置主线**：
- 高端消费：奢侈品（2026年2月瑞士表出口数据跟踪）
- 大众消费：性价比品牌
- 出海消费：中国品牌全球化

## 五、中金分析框架与方法论（可学习借鉴）

### 5.1 宏观-策略-行业三层联动框架

```
第一层：宏观定位（增长/通胀/政策）
    ↓ 传导机制
第二层：策略选择（风格/配置/主题）
    ↓ 落地标的
第三层：行业比较（供需/格局/估值）
    ↓ 精选个股
个股研究
```

**中金宏观到策略的传导路径**：
```
宏观假设（如：实际GDP回升，PPI触底）
    ↓
盈利预测上调（去库存结束→补库存）
    ↓
ERP（股权风险溢价）下降
    ↓
DDM分子（盈利）上修 → 分母（风险偏好）改善
    ↓
A股/港股上行 → 顺周期/成长占优
```

### 5.2 中金政策研究方法

**政策传导"四问法"**：

| 问题 | 内容 |
|------|------|
| ① 政策目标是什么？ | 稳增长/防风险/调结构 |
| ② 政策工具是什么？ | 财政/货币/监管/产业 |
| ③ 传导路径是什么？ | 央行→银行→实体 / 财政→项目→产业链 |
| ④ 效果何时验证？ | 3个月/6个月/1年 |

### 5.3 中金估值框架

**中金DCF修正框架（行业差异）**：

| 行业类型 | 关键假设 | 估值重点 |
|---------|---------|---------|
| 成长型（科技/新能源）| 营收增速/市占率 | PS、EV/S |
| 周期型（化工/钢铁）| PPI/产能利用率 | PB、EV/EBITDA |
| 防御型（消费/医药）| 永续增长率 | PE、DCF |
| 价值型（银行/地产）| 资产质量/拨备 | PB、RNAV |

### 5.4 中金财务预测方法

**盈利预测"三步法"**：
```
第一步：收入端
  - 行业增速 × 市占率变化 = 收入增速
  - 价格弹性（对周期品）

第二步：成本端
  - 毛利率趋势（规模效应/原材料）
  - 费用率管控（管理费用/销售费用）

第三步：资产负债表
  - 营运资本变化（应收账款/存货）
  - 资本开支节奏（决定自由现金流）
```

### 本周最新观点1：美元流动性冲击主导，安全资产定义重构
**来源：《宏观探市4月报：寻找安全资产》| 于文博、张峻栋、张文朗 | 2026-04-03**

- **核心矛盾：** 本轮美以伊冲突造成"流动性需求>避险需求"，传统避险资产（黄金-、美债↓、瑞郎↓）全面失效
- **机制：** 油价飙升→通胀预期抬升→降息预期收窄→宏观流动性预期收紧；离岸美元需求异常上升（贸易结算+融资对冲+避险储备+金融合约追保）
- **结果：** A股(沪深300-5.5%)和标普500(-5.1%)相对抗跌；欧洲(-8%~-19%)和日韩(-13%~-19%)大幅下跌
- **人民币资产**在四次地缘冲击中均呈正超额收益，A股"安全资产"（科技硬件、电力设备、工业材料、能源）超额收益从负转正
- **政策含义：** 全球"安全资产"正从避险资产（黄金/美债）转向**资源能源+科技制造**

**情景分析：**
- 情形A（冲突升级）→ 倒逼美联储扩表→ 短空长多，利好中国股市+金银铜
- 情形B（冲突缓和）→ 流动性压制解除→ 金银铜同受益
