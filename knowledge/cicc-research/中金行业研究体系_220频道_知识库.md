## 二、中金行业研究框架方法论

### 2.1 中金行业分析标准流程

```
【第一步】行业界定与周期定位】
  - 行业所处周期：导入期/成长期/成熟期/衰退期
  - 供需格局：供给弹性 vs 需求弹性
  - 产能周期：扩张期 vs 出清期

【第二步】竞争格局分析】
  - 市场集中度（CR3/CR5）
  - 竞争壁垒：高转换成本/网络效应/规模经济
  - 波特五力：供应商/买方议价/新进入者/替代品/行业内竞争

【第三步】政策环境评估】
  - 鼓励/限制政策
  - 政策传导时滞
  - 受益/受损标的识别

【第四步】财务验证】
  - 营收增速 vs 行业增速（市场份额变化）
  - 毛利率趋势（护城河在财务上的体现）
  - 现金流质量（盈利真实性）
  - 资产质量（资产负债表健康度）

【第五步】估值与配置】
  - 横向比较（vs 行业平均/历史中枢）
  - 纵向比较（vs 历史估值分位）
  - 风险收益比（上行空间 vs 下行风险）
```

### 2.2 中金行业配置决策矩阵

| 行业周期位置 | 特征 | 配置策略 |
|------------|------|---------|
| **导入期** | 渗透率<5%，技术路径不确定 | 主题投资，高弹性，止损严格 |
| **成长期** | 渗透率5%-30%，竞争加剧 | 优选龙头，份额集中逻辑 |
| **成熟期** | 渗透率>30%，增速放缓 | 高股息/防御性，现金流为王 |
| **衰退期/出清期** | 产能出清，供给收缩 | 等待供需拐点，配置剩者 |

### 2.3 中金估值方法适用场景

```
行业           首选估值方法        关键指标
─────────────────────────────────────────
消费（成长）    PEG / PS            营收增速、毛利率
消费（成熟）    PE / 股息率         净利润、FCF
周期（化工）    PB / EV/EBITDA      PPI/产能利用率
金融（银行）    PB / ROE            不良率、拨备覆盖率
科技（硬件）    PS / EV/S           市场份额、研发费用率
电力（公用）    股息率 / EV/EBITDA  利用小时数、电价
房地产          PB / RNAV            NAV折让率、去化率
```

**中金核心判断：AI算力需求爆发 → 光互联从"可选项"变为"必选项"**

**数据支撑**：
```
OFC（光纤通信展）2026：
- AI驱动光模块需求从100G/400G向800G/1.6T加速升级
- 数据中心内部流量爆发 → 光互联（DAC/AOC/光模块）量价齐升
- 中国光通信产业链：华为/中兴/中际旭创/新易盛 具备全球竞争力
```

**中金投资逻辑**：
```
需求侧：AI训练集群扩张 → 光互联带宽需求5年5倍
供给侧：中国厂商占全球光模块产能 ~70%

→ 结论：超配光通信龙头
```
