## 核心摘要

**主题**：新冠疫情 + 俄乌冲突叠加，中国经济面临**三重压力**：需求收缩、供给冲击、预期转弱

**核心判断**：
- 滞胀不是短期问题，是结构性长期压力
- 中国需从"信贷扩张"向"财政扩张"转型
- 最优政策组合：**紧信用、松货币、宽财政**

---

## 一、短期：疫情与战争的滞胀压力

**疫情三条传导路径：**
1. 国际贸易 → 贸易量跌、价格涨 → **典型滞胀**（2020年）
2. 投资不足 → 产能增长受限 → 供给恢复慢于需求
3. 贫富分化加剧 → 低收入群体消费力下降

**俄乌冲突影响：**
- 能源 + 粮食供给缺口 → 加剧全球滞胀
- 俄乌小麦出口占全球25%，埃及80%小麦依赖两国
- 历史经验：粮食危机 → 发展中国家社会动荡

---

## 二、中长期：四大供给约束

### 1. 人口老龄化
- 中国劳动年龄人口即将负增长
- 美国相对健康：移民补充年轻劳动力
- **中国需吸引周边国家年轻移民**

### 2. 碳中和（绿色转型）
- 化石能源成本上升 → 社会生产成本系统性抬高
- 三种路径：技术进步、碳定价、社会治理
- 欧盟碳市场价格已趋势性上升
- 中国高碳欠发达地区人均GDP差距持续扩大

### 3. 产业链调整（横向+纵向）
- **横向（去中心化）**：西方担心过度依赖中国 → 制造业外迁 → 成本增加
- **纵向（卡脖子）**：中国处于全球产业链下游 → 自主创新成本大幅上升

### 4. 和平红利消退
- 1960年代 OECD军费占GDP 6-7%
- 冷战结束后降至3%以下（和平红利）
- 俄乌冲突 → 德国等欧洲国家军费重启上升
- 全球滞胀新增供给侧压力

## 三、区分两类债务：财政 vs 信贷

| | 财政扩张 | 信贷扩张 |
|---|---|---|
| **货币投放** | 基础货币（财政赤字） | 派生货币（银行信贷） |
| **接收方** | 私人部门净资产增加 | 负债增加，不增加净资产 |
| **主要受益** | 消费（低收入群体） | 投资+资产价格（房地产） |
| **历史对应** | 通胀（六七十年代） | 资产泡沫+债务危机 |
| **当前案例** | 美国（通胀） | 中国（债务+地产风险） |

**关键洞察**：
- 美国：财政扩张 → 居民净资产↑ → 消费强 → 通胀压力
- 中国：信贷扩张 → 地产泡沫 + 债务风险 → 金融周期下行
- **中国需从信贷向财政转型**

## 四、宏观政策框架："紧信用、宽财政、松货币"

### 政策优先级（财政扩张三部曲）
1. **减税降费** → 避免企业倒闭，促进供给
2. **转移支付** → 帮扶低收入群体
3. **基建** → 最后选项（供给不足时易引发通胀）

### 资金来源
- 政策性金融（准财政）：普惠金融、绿色信贷、科技创新再贷款
- **央行上缴利润**：财政+货币协同新篇章 = 基础货币直接支持财政

### 2022年广义财政预算
- 广义财政收支差额 ≈ **GDP的9.2%**
  - 6.1% 来自发债（国债+地方债）
  - 3.1% 来自基础货币增加

### 货币政策重心
- 在美联储加息背景下，降息空间受限
- 人民币汇率仍在高位
- **结构性工具重要性上升**：绿色转型、科技创新再贷款
