## 一、金句（直接可引用的判断）

**关于中国经济核心矛盾**：
> "中国宏观经济最简洁的描述，就是内部有效需求不足。"
> "不是信心不足，是口袋里没钱。刺激消费没用，有钱自然会花。"

**关于中产阶级**：
> "中产阶级是有效需求的核心来源，不是顶层（边际消费倾向低），也不是底层（无支付能力）。"
> "这两年网约车司机突然激增两千万——这些人是被裁员的白领（中产）。"

**关于债务与信贷**：
> "美国靠财政发钱给居民，中国靠信贷推地产——结果完全不同。美国居民净资产上升，中国债务风险暴露。"

**关于老龄化**：
> "老龄化带来的任何产业增长，都将弱于年轻人消费总量消退的数字。"

**关于2016年后世界观**：
> "金字塔顶层是意识形态，社会政治的周期本质上是左右翼的更替。2016年后全球右翼化，逻辑需要重建。"

**关于AI与2026年**：
> "这一轮不是短周期的反弹，而是三个齿轮（生产力-生产关系-制度秩序）的历史性重构。2026年是关键分水岭。"

## 二、核心分析框架速查

### 框架1：三大核心问题（任何宏观经济分析的统一入口）
```
分工 & 分配  →  财富创造
债务 & 杠杆  →  财富扩张
收入 & 贫富  →  财富分配
```

### 框架2：金字塔分析（自上而下看资产）
```
意识形态（左/右）  →  决定政策取向
政治/地缘          →  决定宏观环境
市场/资产价格      →  最终落地表现
```

### 框架3：三齿轮理论（2026年判断核心）
```
生产力（AI/技术） → 拉动 → 生产关系（制度） → 重塑 → 制度秩序
```

### 框架4：中美比较（判断大类资产的关键）
```
美国：财政主导  → 通胀风险  → 居民净资产↑  → 消费强
中国：信贷主导  → 债务风险  → 地产依赖     → 有效需求↓ 
```
