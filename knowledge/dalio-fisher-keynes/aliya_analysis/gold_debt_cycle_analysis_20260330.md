## 一、核心发现

### 1.1 达利欧最新警告：美国债务危机可能3年内到来

达利欧在2025年6月发表的文章《国家如何破产：大周期》中明确警告：

> **"如果我们的路线没有改变，它（美国债务危机）将在三年内到来，大约相差不超过两年。"**

#### 关键数据

| 指标 | 数据 | 评估 |
|------|------|------|
| 美国国债总额 | 39万亿美元 | 历史新高 |
| 债务/收入比 | 6.5-7倍 | 不可持续 |
| 年利息支出 | 超1万亿美元 | 每4美元收入1美元还息 |
| 10年后预测债务 | 50-55万亿美元 | 约占GDP 300%+ |

#### "3%三条腿走路"策略

达利欧提出的解决方案：

```
削减支出 → 增加收入 → 印钞（最后手段）
```

**关键判断**：如果能将赤字占GDP比重从7%降至3%左右，风险将大大降低。

## 四、美元债务周期分析

### 4.1 大债务周期理论框架

达利欧的大债务周期通常持续约80年，分为五个阶段：

| 阶段 | 特征 | 美国当前状态 |
|------|------|-------------|
| **1. 健全货币** | 低利率，借债回报>资本成本 | ❌已过 |
| **2. 债务泡沫** | 债务扩张，资产价格上涨 | ❌已过 |
| **3. 泡沫破裂** | 债务扩张未停，资产价格达顶 | ⚠️接近 |
| **4. 去杠杆化** | 债务违约，资产暴跌，利率降至零 | ⏳进行中 |
| **5. 债务危机** | 债务货币化，新周期开始 | ⏳倒计时 |

### 4.2 美国当前所处阶段

**判断**：美国正处于第3-4阶段过渡期

关键指标：
- 债务/GDP：125%+（不可持续）
- 利率：降至3.75%-4.00%（2025年三次降息）
- 货币：美元指数2025年下跌9%（8年最差）
- 政治：中期选举临近（2026年）

### 4.3 债务危机触发条件

达利欧指出，当以下情况发生时危机将到来：

```
债务融资支出紧缩 = 经济心脏病发作
         │
         ├── 无法展期债务
         ├── 债务拍卖失败
         ├── 利率急剧上升
         └── 货币大幅贬值
```

## 六、结论

### 6.1 核心结论

1. **黄金战略地位上升**：正在取代美债成为避险首选
2. **美元债务不可持续**：3年内可能迎来危机
3. **杠铃策略有效**：AI+黄金的配置兼顾收益与风险
4. **去美元化加速**：美元储备占比持续下降

### 6.2 Aliya建议

**战略层面**：
- 增加黄金配置至5%-15%
- 减少美债敞口，尤其是长久期
- 增加非美资产配置

**战术层面**：
- 关注美国债务上限谈判
- 监测央行购金动态
- 警惕AI股票估值回调

**长期视角**：
- 以大周期理论审视当前格局
- 做好应对"失序时期"的准备
- 坚持风险平价和全天候策略
