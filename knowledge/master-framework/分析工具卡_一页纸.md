## 🔍 第一步：判断宏观周期（这一步决定一切）

### 达利奥债务周期定位

```
偿债负担 = 还本付息 / 收入
债务/GDP = 总债务 / 当年GDP
短期债务占比 = 短期债务 / 总债务

>18% + >300% + >50% → 去杠杆阶段 → 防御配置
<14% + <200% + <30% → 扩张阶段 → 权益进攻
```

### 费雪实际利率

```
实际利率 = LPR - PPI同比
> 3% → 实质紧缩
0-3% → 中性
< 0% → 实质宽松 ← 当前？（查当月数据）
```

### 凯恩斯总需求

```
GDP缺口 = 潜在GDP - 实际GDP
缺口<0 → 扩张政策必要 → 财政/货币双宽松
缺口>0 → 收缩政策必要 → 财政/货币收紧
```

## 📊 第二步：产业赛道选择

### 中金行业配置矩阵

```
景气↑ + 政策支持 → 超配（光通信/机器人/AI）
景气↑ + 政策受限 → 标配（钢铁/煤炭）
景气↓ + 政策支持 → 标配（房地产/城投）
景气↓ + 政策受限 → 低配（民营开发商/高杠杆）
```

### 波特五力快评（5分钟）

```
供应商议价：低/中/高？
买方议价：低/中/高？
新进入者威胁：高/低？
替代品威胁：高/低？
行业内竞争：激烈/温和？
→ 得出护城河宽窄的初步判断
```

## 💰 第三步：个股估值三法

### 格雷厄姆安全边际

```
PE(TTM) ≤ 20 ✅
PB ≤ 1.5 ✅（或<3但ROE>15%）
连续10年无亏损 ✅
连续20年支付股息 ✅
EPS10年增长 ≥ 33% ✅
```

### 巴菲特护城河4问

```
① 客户为什么选你而非竞争对手？（核心优势）
② 10年后这道护城河还在吗？
③ 竞争对手复制有多难？
④ 管理层把股东的钱当自己的吗？
```

### DCF三假设

```
① 所有者收益增速：乐观/基准/悲观
② 折现率：9%/10%/11%
③ 永续增长率：6%/4%/2%
→ 三种情景下的内在价值区间
→ 当前股价 vs 内在价值 → 安全边际
```

## ⚠️ 第四步：风险管理

### 芒格逆向清单

```
□ 最可能失败的3个原因？
□ 如果错了，什么信号告诉我？
□ 下跌50%我能承受吗？
□ 5年股市关门，这投资还合理吗？
□ 有没有更简单的赚钱方式？
```

### 仓位上限

```
高确定性机会 → 组合5-10%
中等确定性机会 → 组合2-5%
低确定性/试探性 → 组合≤2%
单一标的 → 不超过组合15%
```
