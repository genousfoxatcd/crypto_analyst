## 第二部分：宏观周期定位工具箱

### 2.1 达利奥债务周期框架（定位工具）

**使用步骤**：

**第一步：计算核心指标**

```
偿债负担 = 还本付息额 / 收入
债务/GDP = 总债务存量 / 当年GDP
债务结构 = 短期债务 / 总债务
```

**第二步：对照债务周期位置**

| 阶段 | 偿债负担 | 债务/GDP | 政策应对 | 资产表现 |
|------|---------|---------|---------|---------|
| 早期扩张 | <14% | <200% | 宽松 | 权益涨 |
| 中期扩张 | 14-18% | 200-300% | 温和收紧 | 分化 |
| 去杠杆初期 | >18% | >300% | 收紧+违约 | 债券跌/权益跌 |
| 货币化/再通胀 | 触顶 | 平台期 | 央行QE | 权益反弹 |
| 新扩张 | 下降后回升 | 重新积累 | 宽松 | 全面涨 |

**第三步：输出结论**

```
当前位置：[填入]
核心矛盾：[填入]
距上次去杠杆化峰值：X年
当前政策措施：[货币主导/财政主导/混合]
向下一阶段演变的关键信号：[3个指标]
```

### 2.2 凯恩斯总需求分析框架（政策工具）

**总需求缺口测算**

```
潜在GDP（供给侧估算）= 劳动×资本×全要素生产率
实际GDP缺口 = 潜在GDP - 实际GDP
缺口为负 → 经济低于潜在产出 → 扩张性政策必要
缺口为正 → 经济过热 → 紧缩性政策必要
```

**凯恩斯乘数应用**

```
财政乘数 K = 1 / (1 - MPC)

MPC（边际消费倾向）估算：
- 中国居民MPC ≈ 0.65（基于历史数据）
- K ≈ 1/(1-0.65) = 2.86
- 即财政支出100亿 → GDP增加286亿（含间接效应）

注意：
- 产能过剩时乘数效应更强（闲置资源被利用）
- 产能饱和时乘数效应减弱（资源约束）
- 流动性陷阱时乘数失效（货币宽松无效）
```

**IS-LM 框架应用**

```
政策组合选择：

情况A：经济衰退 + 利率已降至零
  → 凯恩斯流动性陷阱解法：财政政策主导
  → 货币政策无效，必须靠财政

情况B：经济衰退 + 利率仍有空间
  → 财政+货币双宽松组合
  → 先降息（传导快），再配合财政（效果持久）

情况C：经济过热 + 通胀压力
  → 先货币收紧（抑制需求）
  → 再配合产业政策（抑制过剩产能）

情况D：滞胀（低增长+高通胀）
  → 最难处理，供给侧改革优先
  → 财政收紧（抑制总需求）+ 产业政策（增加供给）
```

### 2.3 费雪实际利率框架（货币工具）

**实际利率计算**

```
实际利率 = 名义利率 - 实际通胀率
        = 1年期LPR - PPI同比
        = 或：5年期LPR - GDP平减指数

判断标准：
- 实际利率 > 3%：实质紧缩（借贷成本高）
- 实际利率 0-3%：中性区间
- 实际利率 < 0%：实质宽松（借贷成本极低）
```

**实际利率 → 信贷需求传导**

```
实际利率下行 + 企业盈利改善 → 信贷需求回升 → 社融回暖 → 经济上行
实际利率上行 + 企业盈利恶化 → 信贷需求萎缩 → 社融下滑 → 经济下行

关键传导指标（月度跟踪）：
- 社会融资规模存量同比
- M2同比
- M1同比（企业活化资金）
- LPR报价（1年和5年）
```

### 2.4 黄奇帆地方视角（政策传导节点）

**地方政府的"重组"操作节点**

```
中央政策出台
    ↓ 传导时滞：1-3个月
省级政府制定实施方案
    ↓ 传导时滞：2-6个月
地市政府细化落实
    ↓ 传导时滞：3-12个月
企业/居民行为响应
    ↓ 效果显现
经济数据变化
```

**黄奇帆"重组"框架在政策传导中的应用**

```
重组工具箱：
① 边界调整（中央vs地方事权划分）
② 资源配置（专项债额度分配）
③ 利益分配（税收分成比例调整）
④ 结构调整（产业目录/准入门槛调整）
⑤ 管理政策（审批流程优化/监管宽严）

判断政策执行力度的黄奇帆三问：
1. 地方有没有足够的财政资源落实？（资金来源）
2. 地方政府有没有足够的激励去落实？（政绩导向）
3. 有没有足够的问责机制确保落实？（督查体系）
```

## 第三部分：产业分析框架

### 3.1 中金行业配置三层联动模型（产业选择标准流程）

**第一层：宏观假设**

```
GDP增长假设：X%
PPI走势：上行/平台/下行
货币政策：宽松/中性/收紧
财政政策：扩张/中性/收缩
汇率：升值/稳定/贬值
```

**第二层：策略配置方向**

```
经济上行期：
  → 顺周期行业（有色/化工/钢铁）
  → 利率敏感型（银行/券商/保险）
  → 消费可选（汽车/家电/旅游）

经济下行期：
  → 防御型（食品/医药/公用事业）
  → 高股息（电力/运营商/银行）
  → 逆周期（基建/电力设备）

政策刺激期：
  → 政策受益（新能源汽车/半导体/光伏）
  → 产业基金投资方向（"十五五"规划）
```

**第三层：行业比较矩阵**

```
横轴：行业景气度（向上/向下/平稳）
纵轴：政策确定性（政策支持/中性/受限）
气泡大小：行业市值/占比

→ 落在右上象限（景气向上+政策支持）= 超配
→ 落在左下象限（景气向下+政策受限）= 低配
```

### 3.2 波特五力竞争分析模板

```
供应商议价能力（高→行业利润率受压）
  - 供应商数量（集中还是分散）
  - 替代供应商是否存在
  - 供应商转换成本

买方议价能力（高→毛利率受压）
  - 买方集中度
  - 产品差异化程度
  - 买方转换成本

新进入者威胁（高→行业回报率下降）
  - 规模经济壁垒
  - 资本壁垒
  - 品牌壁垒
  - 渠道壁垒

替代品威胁（高→需求被侵蚀）
  - 替代品性价比
  - 用户转换意愿
  - 技术迭代速度

行业内竞争（激烈→价格战）
  - 竞争者数量和集中度
  - 行业增速（增量vs存量市场）
  - 退出壁垒高低
  - 产品差异化程度
```

### 3.3 产能周期定位框架

```
产能周期四阶段：

阶段一：产能出清（底部）
  - 产能利用率降至历史低点
  - 行业普遍亏损
  - 龙头企业开始限产
  - 典型信号：PPI同比持续为负

阶段二：供需紧平衡（拐点）
  - 需求率先回升
  - 产能利用率回升
  - 库存去化完成
  - 典型信号：PPI同比由负转正

阶段三：产能扩张（上行）
  - 行业盈利改善
  - 龙头企业率先扩产
  - 新进入者增加
  - 典型信号：固定资产投资增速回升

阶段四：产能过剩（顶部）
  - 产能扩张快于需求
  - 价格战开始
  - 行业盈利下滑
  - 典型信号：PPI同比由正转负，库存积压
```

## 第四部分：投资估值框架

### 4.1 格雷厄姆安全边际体系

**格雷厄姆选股7条**

```
1. 企业规模（不能是微型股）
   → 营业收入 > X亿元（根据行业调整）

2. 财务状况（流动比率 ≥ 2）

3. 盈利稳定性（连续10年无亏损）

4. 股息记录（连续20年支付股息）

5. 盈利增长（EPS过去10年增长 ≥ 33%）

6. 估值控制（PE ≤ 20）

7. 股价/净资产（PB ≤ 1.5）
```

**格雷厄姆公式（动态调整版）**

```
内在价值 = EPS × (8.5 + 2g)

调整版（适应低利率环境）：
内在价值 = EPS × (8.5 + 2g) / (当前AAA债券收益率/4%)

其中：
- g = 未来7-10年预期增长率（%）
- 当前10年国债收益率作为无风险利率基准
```

**净流动资产法（NCAV）**

```
NCAV = 流动资产 - 全部负债
股价 < NCAV × 0.67 → 极度低估（格雷厄姆"烟蒂"标准）
股价 < NCAV × 0.85 → 相对低估
```

### 4.2 巴菲特护城河评估4问

```
问题1：客户为什么选择这家公司而非竞争对手？
   → 找出最核心的竞争优势（品牌/成本/粘性/网络效应）

问题2：这道护城河10年后还在吗？
   → 评估竞争优势的持续性
   → 技术变革/政策变化/颠覆性替代品威胁

问题3：竞争对手复制这道护城河有多难？
   → 规模经济/专利/网络效应/转换成本

问题4：管理层能否把股东的钱当作自己的钱？
   → 资本配置决策（回购/分红/再投资）
   → 激励机制与股东利益的一致性
```

### 4.3 DCF估值标准流程

**第一步：估算所有者收益**

```
所有者收益 = 净利润 + 折旧摊销 - 必要资本支出

必要资本支出判断：
- 维持性资本支出（维持现有竞争地位）
- 扩张性资本支出（扩大规模）
→ 区分两者是准确DCF的关键
```

**第二步：确定折现率**

```
折现率 = 无风险利率 + 股权风险溢价

无风险利率：中国10年国债收益率
股权风险溢价：A股历史ERP ≈ 5-7%

折现率范围：
- 稳健保守：10年国债 + 6% ≈ 9-10%
- 合理基准：10年国债 + 5% ≈ 8-9%
- 乐观情景：10年国债 + 3% ≈ 6-7%
```

**第三步：情景设置**

```
永续增长率（g）的设定规则：
- 不能超过GDP名义增速（长期约6-8%）
- 成熟行业g通常取2-3%
- 成长行业g可取5-8%，但需要极强护城河支撑

三种情景：
- 乐观：g=6%，永续
- 基准：g=4%，永续
- 悲观：g=2%，永续
```

### 4.4 芒格逆向检验清单

```
决策前必问：

□ 这笔投资最可能失败的原因是什么？（列出3个）
□ 如果这个逻辑是错的，什么信号会告诉我？
□ 我有没有因为"喜爱偏差"高估这家公司？
□ 我有没有因为"社会认同"跟随了他人判断？
□ 如果我的配偶/朋友反对这项投资，我的理由还成立吗？
□ 我是否在赌"这次不一样"？（永远不要这样赌）
□ 这笔投资在"5年内股市关闭"的前提下还合理吗？
□ 我是否能承受这笔投资下跌50%而不恐慌？
□ 这个机会的预期收益率是否值得我承担的风险？
□ 有没有更简单、更确定的赚钱方式？（通常是）
```

## 第五部分：政策分析框架

### 5.1 政策传导四问法

```
问题1：政策目标是什么？
  → 稳增长/防风险/调结构/惠民生
  → 单一目标还是多重目标？（多重目标=权衡取舍）

问题2：政策工具是什么？
  → 财政（税收/支出/转移支付）
  → 货币（利率/存款准备金率/流动性）
  → 监管（窗口指导/红线管理）
  → 产业（补贴/准入/目录）

问题3：传导路径是什么？
  → 直接传导（政策→企业/居民）
  → 间接传导（政策→金融机构→实体经济）
  → 时间滞后：1个月/3个月/6个月/12个月

问题4：效果验证节点是什么时候？
  → 高频数据（1-3个月）：社融/PMI/工业增加值
  → 中频数据（3-6个月）：GDP/CPI/企业盈利
  → 低频数据（6-12个月）：就业/居民收入/财政收支
```

---

### 5.2 黄奇帆政策评估三维度

```
① 资金从哪里来？（财政/货币/社会资本）
  → 没有资金来源的政策是空政策
  → 专项债额度？特别国债？PSL？

② 执行激励机制是什么？
  → 地方政府的政绩导向是否与政策目标一致？
  → 问责机制是否清晰？

③ 政策退出机制是什么？
  → 没有退出机制的政策会产生依赖
  → 补贴退坡路径？退出时间表？
```
