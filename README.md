## 分析框架

采用**超级马里奥五要素分析法**：
1. **价格锚定** — 当前价格 vs 历史分位
2. **供需判断** — 供给侧 + 需求侧驱动因子
3. **政策催化** — 国内政策 + 海外联动
4. **资金流向** — ETF 资金流 + 持仓变化
5. **配置建议** — 仓位比例 + 入场条件 + 止损线
